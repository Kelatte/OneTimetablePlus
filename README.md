### 使用教程

  编辑界面的配置方法
  - **左键**右侧课程名称按钮，可在选中的日子末尾加入该课程。注意每天最多有**9**节课。
  - **右键**右侧课程名称按钮，可删除该课程
  - 点击下方按钮**删除该课**，可删除左侧选中的单节课程
  - 新加课程种类步骤
    1. 有两个文本框，在上方的输入课程全称（如 语文）
    2. 在下方的输入课程简称（如 语）
    3. 点击**新加课种**
  - 循环课表使用方法
    - 点击**新加当日循环课表**，可在选中星期几添加一个会每周自动循环切换课表的多重课表，以实现A、B……表要求。若当前选中的为普通课表（即不会循环的）那么会隐藏该普通课表
    - 点击**删除该循环课表**，可删除一天的循环课表。若当日循环课表删完了，那么原先的普通课表就会显示出来
  - 修改时会实时更新在显示窗口中。修改完后记得**保存**。若误修改可点击**重载**返回至上一保存状态。
